Git — ключовий інструмент в сучасній розробці, що дозволяє працювати спільно та відстежувати зміни.
При виконанні завдання я налаштував SSH-доступ, створив гілки та зробив pull request-и.
Найзручнішим виявився інтерфейс GitHub для PR, який наочно показує відмінності.
Найскладнішим було розібратися з помилками доступу по SSH та вирішити конфлікти при push.
Здобуті навички допоможуть організувати роботу в команді при виконанні курсових і групових проєктів.
